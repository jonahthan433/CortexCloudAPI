# Security Policy

CortexCloud takes the security of our AI Gateway and developer credentials seriously.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v0.1.x  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in CortexCloud, please **do not** open a public GitHub issue. Instead, report it privately to our security team via email at `security@cortexcloud.ai`.

Please include:
*   A description of the vulnerability.
*   Step-by-step instructions or a proof-of-concept script to reproduce the issue.
*   The potential impact of the exploit.

We will acknowledge receipt of your report within 24 hours and provide weekly updates on our progress toward a fix. We follow a coordinated vulnerability disclosure process.
